About Me

I'm a self-taught developer and I've been playing around with Python and JavaScript for several years now. From writing small scripts for parsing through financial market data; to coding along with courses and building small e-commerce websites; to building my own full-stack, containerized application hosted on Heroku. Initially I had a lot of fun playing with API's for financial market data and sports statistics, and even though I was limited in what I could build the possibilities we're very clear to me and very enticing. I reunited with old friends who are more experienced programmers but equally motivated and began collaborating on more serious projects. Now, as I've developed my skills across a couple of different languages and frameworks, I've grown more confident in my ability to conritbute to a team in a professional role. Earning credibility and a reputation in the tech industry, while also contributing to meaningful work and increasing my earning potential are my main focus.

In preparation for this career transition I've gone more in depth into data structures and algorithms so I can contribute meaningfully to discussions about implementation and architecture, and also to be more confident that I can consistently implement optimal solutions. Admittedly, I've taken a longer time than I probably should have to begin exploring careers in this space, but I'm excited to finally start building interesting projects and solve engaging problems.

Projects

Fantasy Football Draft Board

Check out a beta version of the app!

This project is a tool for ranking and organizing NFL players for fantasy football drafts. I want to build something that I can actually use even if the quality isn't the best, and something that I can't readily find online the way I would like to use it.

The App features user accounts and incorporates a basic login and authorization system so that users can save their changes. The UI also offers the option to drag-and-drop players around the draftboard (needs some refining on mobile) for a smoother experience. Features I plan on adding incrementally include autosave, profile customization, and a notes section for players. I suppose I could show the visual design some more love too...

Please reach out to provide any feedback on bugs, UI, and code structure if you decide to engage with the app!

React Front End with Redux

The front-end is built with React and uses Redux for state management. It incorporates many modern features and common design patterns like hooks and functional components. Sorry about the design, but I focused on the back-end and plan on improving the UI gradually.
Check out the Github repo for the front end!

Node Server with Express and a PostgreSQL DB

The back end is built with Node.js and the Express framework. The main database uses PostgreSQL, and the app also incorporates Redis for session management.
Check out the Github repo for the back end!

Containerized with Docker & Hosted on Heroku

The server lives in a Docker container hosted on Heroku with its own dedicated Heroku databases.
